Load posts/products via Ajax with customizable settings.

== Description ==

Universal Ajax Loader allows you to load posts and products dynamically via Ajax without page refresh. Perfect for creating modern, fast-loading websites with smooth user experience.

**Features:**
* Ajax loading for any post type
* Customizable posts per load
* Infinite scroll option
* Multiple layout options
* Easy shortcode implementation
* Professional admin interface

**Usage:**
Simply use the shortcode `[ajax_post_loader]` in any post or page.

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/universal-ajax-loader/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Settings > Ajax Loader to configure
4. Use the shortcode `[ajax_post_loader]` in your content

== Frequently Asked Questions ==

= How do I use this plugin? =
After activation, go to Settings > Ajax Loader, configure your settings, and use the shortcode `[ajax_post_loader]` anywhere you want Ajax loading.

= Does it work with custom post types? =
Yes, you can select any public post type from the settings.

= Is it compatible with WooCommerce? =
Yes, it works with WooCommerce products and any other custom post types.
